CSS gehen im Inline, Internal und External

Hinter dieser Webseite verbirgt sich ein Code im style.css Sheet im VS Code.

Außerdem spiele ich etwas mit den Formattierungen wie margins, paddings, borders usw.

- Internal sind css Formate, die im öffnenden Tag geschrieben werden. Bsp: < div style="color: white;" usw...
- Internal ist das css Format nach dem Header als style Tag.
- External ist das css welches über ein neues Dokument als Spreadsheet: style.css in die .html-Datei verlinkt wird.
Bsp: < link rel="stylesheet" href="style.css"
